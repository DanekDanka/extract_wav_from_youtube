# extract_wav_from_youtube
This is the application for saving wav files from toutube videos

Это приложение для скачивания wav файлов из видео с youtube

---
Для установки склонируйте репозиторий:
```
git clone https://github.com/DanekDanka/extract_wav_from_youtube.git && cd extract_wav_from_youtube
```

Установите виртуальное окружение:
```
python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
```
Запустите приложение:
```
python3 audio_dataset_creator.py
```
